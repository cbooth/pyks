pyks.get_current_ks()

Returns the current kilosecond time as a float integer.

Takes no arguments.

Example code (Python 3.2):

import pyks
import time

local_time = time.localtime()
print('The current time is:', time.strftime('%H:%M:%S',local_time))
print('In kiloseconds, this is: {0}'.format(get_current_ks()))

pyks.time_to_ks(local_time)

Converts a struct_time object into a kilosecond value.

Returns as a float integer.

Takes exactly one argument:

  • local_time - The struct_time object to convert

Note: Argument local_time's naming is due to the fact that it is an internally used function in the get_current_ks() function, although it can be used externally.

Example code (Python 3.2):

import pyks
import time

print(pyks.time_to_ks(time.localtime()))

pyks.ks_to_human(kilosecond)

Takes a kilosecond value and converts it to a human-readable HH:MM:SS time string

Returns as string

Takes exactly one argument:

  • kilosecond - The value to convert to an HH:MM:SS time string

Note: The same effect can be acheived using pyks.format_ks(ks, format_ks) and setting the argument 'format_ks' to '%H:%M:%S'

Example code (Python 3.2):

import pyks

print(pyks.ks_to_human(42))

pyks.format_ks(ks, format_ks)

Formats a kilosecond value using the format_ks argument.

Takes exactly two arguments:

  • ks - The kilosecond as a float integer.
  • format_ks - The string to format the ks against.
format_ks usage

The format_ks argument takes the same type of argument as the time.strftime() function. Where:

  • '%H' corresponds to the hours section of the time string.
  • '%M' corresponds to the minutes section of the time string.
  • '%S' corresponds to the seconds section of the time string.

These components can be assembled in any way to create a time string from a ks value

Example code (Python 3.2):

import pyks

# Define some time strings

hhmmss = '%H:%M:%S'
hhmm = '%H:%M'
mmss = '%M:%S'
# Other symbols can be used in between
ssmmhh = '%S-%M-%H'

# Add the current ks to a variable

thisks = pyks.get_current_ks()

# Format the ks

print(pyks.format_ks(thisks, hhmmss))
print(pyks.format_ks(thisks, hhmm))
print(pyks.format_ks(thisks, mmss))
print(pyks.format_ks(thisks, ssmmhh))
