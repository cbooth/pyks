pyks by cbooth

Python kilo-seconds manipulation library.

Pyks is a kilosecond manipulation module written in Python 3.2 It can be used to convert HH:MM:SS to a kilosecond value and back again, as well as converting the current time into it. Created by Callum Booth and licensed under a Creative Commons Attribution-ShareAlike 3.0 Unported License. Based on the original concept and code from bavardage on GitHub: https://github.com/bavardage/Kiloseconds.git

Install

Run 'setup.py install' at a terminal or command prompt.

Testing

Pyks comes with a set of unit tests written by iPope on GitHub. These tests are stored in ks_tests.py and can be ran from a terminal or command prompt. If the tests report errors, download Pyks again and re-install. Failing that, email me explaining the problem with a copy of the output.
